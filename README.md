Just Another Calendar Widget

I wanted to bring desktop widgets back, so I created Just Another Calendar Widget.
This calendar widget displays a 4 month calendar or quadrimester view, which is the most superior view, IMO.
This widget is designed for a black background which works well with a dark GUI.
The current date is always in the red square.  Holidays are marked in yellow.  Weekends are shaded.
The holiday array only goes though 2028.
